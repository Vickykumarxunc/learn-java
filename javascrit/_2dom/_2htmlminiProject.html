Frontent development is the development of the graphical user interface of a website through the use of HTML CSS JAVASCRIPT so that users can view and interact with that website .

  • HTML : HTML, or HyperText Markup Language, is the standard markup language used to define the structure and meaning of web content. It's a fundamental building block of the web and is used by web browsers to display text, images, and other media on a page.

  • CSS : CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files

    1. Pure CSS

    2. Bootstrap (frameWork)

    3. Tailwind

    4. Sass (preprocessor)


  • JavaScript: JavaScript is a programming language that adds interactivity to web pages. It was originally developed to make web applications more dynamic, allowing browsers to respond to user interactions and change the content layout on a page. Today, JavaScript is used for both client-side and server-side development.

    1. Vallina jS

    2. jQuery (library)

    3. React (framework)

    4. Angular (frameWork)

    5. typeScript

    6. three.js
